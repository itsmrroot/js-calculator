# JS Calculator

A clean, modern calculator built with vanilla HTML, CSS, and JavaScript. Inspired by the iOS calculator design.

## Live Demo

🔗 **[View Live](https://your-username.github.io/js-calculator)**
> Replace the link above with your GitHub Pages URL once published.

---

## Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Decimal point support
- Backspace button to delete last character
- Input validation — prevents invalid sequences like `++` or `..`
- Safe expression evaluation (no `eval`)
- Smooth button press animations

## Tech Stack

- HTML5
- CSS3 (Grid, custom properties, animations)
- Vanilla JavaScript

## Run Locally

No build tools needed. Just clone and open:

```bash
git clone https://github.com/your-username/js-calculator.git
cd js-calculator
open index.html
```

## Deploy to GitHub Pages

1. Push the project to a GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select the `main` branch and `/ (root)` folder
4. Click **Save** — your site will be live at `https://your-username.github.io/js-calculator`
